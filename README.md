# modern_javascript

자바스크립트를 학습한 내용을 아카이빙하기 위한 저장소입니다.
현재는 아래 목차와 같이 같이 레퍼런스 별로 폴더가 구성되어 있습니다.

⚠️ 레퍼런스에서 필요한 부분을 공부하며 정리한 자료이므로 생략된 부분이 있을 수 있습니다.
